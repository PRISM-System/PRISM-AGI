기본 정보

도구의 고유한 이름을 입력하세요. 영문, 숫자, 언더스코어만 사용 가능합니다.
도구의 기능과 용도를 자세히 설명해주세요.

입력 스키마

도구가 받을 수 있는 입력 매개변수를 JSON 스키마 형식으로 정의하세요.
예시:
{
  "type": "object",
  "properties": {
    "city": {
      "type": "string",
      "description": "도시 이름"
    },
    "unit": {
      "type": "string",
      "enum": ["celsius", "fahrenheit"],
      "description": "온도 단위"
    }
  },
  "required": ["city"]
}

엔드포인트 설정

도구가 호출할 API의 전체 URL을 입력하세요.
API 호출에 필요한 헤더를 JSON 형식으로 입력하세요.
API 응답을 기다릴 최대 시간 (1-300초)

1. 도구 이름

JSON Schema 형식으로 매개변수를 정의합니다. type, properties, required 필드를 포함하세요.

3. 엔드포인트

실제 호출될 API의 URL과 HTTP 메서드를 정확히 입력하세요.
